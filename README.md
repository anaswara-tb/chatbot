# NLU Trainer

## Setup

### Clone the repository

```bash
git clone https://github.com/anaswara-tb/nlu_trainer.git
cd nlu_trainer/
```

### Create virtual environment

```bash
# Using conda
conda create -n nlu_env python=3.9.13
conda activate nlu_env
```

### Install dependencies

```bash
pip install -r requirements.txt
```

## Run the app

```bash
python app.py
streamlit run frontend.py
```
